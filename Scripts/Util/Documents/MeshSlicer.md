# MeshSlicer 설명

이 클래스는 **"3D 메시를 평면(칼날)으로 잘라서 두 조각 메시를 만든다"**를 구현하는 정적 유틸리티다.
`ExecutionSliceEffect`가 내부적으로 이 클래스를 호출한다.

---

## 핵심 개념: 평면으로 자르기

```
        평면 (Plane)
           |
   upper   |   lower
   (위쪽)  |  (아래쪽)
     ◇     |     ◇
    ◇ ◇    |    ◇ ◇
   ◇   ◇   |   ◇   ◇
```

평면의 **법선(normal) 방향** 쪽이 upper, 반대쪽이 lower.

---

## 전체 흐름

```
원본 메시 → 정점 분류 → 삼각형 처리 → 단면 캡 생성 → 두 개의 Mesh 반환
```

---

## 주요 구조체

### SlicedMesh (결과물)
- `upperMesh` — 평면 법선 방향 조각
- `lowerMesh` — 반대 방향 조각

### MeshData (작업용 임시 저장소)
메시를 구성하는 데이터를 모아두는 버퍼. 각 조각(upper/lower)마다 하나씩 사용.
- `vertices` — 꼭짓점 위치
- `normals` — 꼭짓점 법선 (조명 계산용)
- `uvs` — 텍스처 좌표
- `triangles` — 겉면 삼각형 인덱스 (submesh 0)
- `capTriangles` — 잘린 단면 삼각형 인덱스 (submesh 1)

---

## 핵심 메서드 설명

### Slice() — 메인 진입점

**Step 1: 정점 분류**
```
각 꼭짓점이 평면의 어느 쪽에 있는지 판별
→ isAbove[i] = true (위) / false (아래)
→ 전부 한쪽이면 자를 수 없으므로 null 반환
```

**Step 2: 삼각형 단위 처리**

메시는 삼각형의 집합이다. 각 삼각형(정점 3개)에 대해:

- **3개 모두 같은 쪽** → 그대로 해당 쪽(upper 또는 lower)에 추가
- **평면과 교차** → `SplitTriangle()`로 분할

**Step 3: 단면 캡 생성**

잘린 교차점들을 모아서 `GenerateCap()`으로 단면을 메워준다.

---

### SplitTriangle() — 삼각형 분할

평면과 교차하는 삼각형은 반드시 **1개 vs 2개**로 나뉜다 (한 꼭짓점만 반대편에 있음).

```
        solo (혼자 있는 쪽)
         /\
        /  \
  ─────/────\───── 평면
      /      \
    pair1   pair2  (같이 있는 쪽)
```

1. **solo 찾기** — 3개 정점 중 혼자 다른 쪽에 있는 정점
2. **교차점 계산** — solo→pair1 에지, solo→pair2 에지가 평면과 만나는 점 (intersect1, intersect2)
3. **solo 쪽**: 삼각형 1개 (solo, intersect1, intersect2)
4. **pair 쪽**: 삼각형 2개 (사다리꼴을 둘로 나눔)

교차점은 `IntersectEdge()`로 계산한다:
- 두 점에서 평면까지의 거리 비율로 보간 위치(t)를 구함
- `Vector3.Lerp(from, to, t)`로 정확한 교차 위치 산출
- 법선과 UV도 같은 비율로 보간 (잘린 부분이 자연스럽게 이어지도록)

---

### GenerateCap() — 단면 뚜껑 만들기

메시를 자르면 내부가 보인다. 이 빈 구멍을 막아주는 것이 **캡(Cap)**.

```
    교차점들을 모아서
    ●───●
   / \ / \    →  부채꼴(fan) 삼각형으로 메움
  ●───●───●
       중심
```

1. **교차점들의 중심** 계산
2. **중복 제거** — 같은 위치의 점 병합 (0.001 이하 거리)
3. **각도 정렬** — 중심에서 각 점까지의 각도로 시계/반시계 정렬
4. **Fan Triangulation** — 중심 + 인접한 두 점으로 삼각형을 부채꼴 모양으로 생성
5. upper에는 법선 방향 캡, lower에는 반대 방향 캡 추가 (양쪽 다 막아야 하니까)

---

### SliceSkinned() — 캐릭터 모델용

`SkinnedMeshRenderer`(뼈대 애니메이션이 있는 모델)를 자를 때 사용.
`BakeMesh()`로 현재 포즈를 정적 메시로 변환한 뒤 일반 `Slice()`를 호출한다.

---

## submesh 구조

결과 메시는 **submesh 2개**로 구성된다:
- **submesh 0** — 원래 겉면 삼각형 → 원본 머티리얼 적용
- **submesh 1** — 잘린 단면 캡 삼각형 → 단면 머티리얼 적용 (빨간색 등)
