# DamageFlash 설명

이 클래스는 **"플레이어가 피격당했을 때 화면 가장자리를 빨갛게 번쩍이게 하는 효과"**를 구현한다.

---

## 전체 흐름

```
게임 시작
  └─ Start()
       ├─ 스프라이트 없으면 → GenerateVignetteSprite()로 비네트 텍스처 자동 생성
       └─ 초기 알파 = 0 (완전 투명)

피격 발생
  └─ DamageFlashListener가 Flash(intensity, duration) 호출
       └─ 타이머 시작

매 프레임
  └─ Update()
       └─ 타이머 감쇠 → flashImage.color.a 줄어듦 → 효과 사라짐
```

---

## 비네트 텍스처 생성 (GenerateVignetteSprite)

512×512 픽셀짜리 이미지를 코드로 직접 그린다.  
픽셀마다 "중심에서 얼마나 먼가"를 계산해서 알파값(투명도)을 정한다.

```
중앙 ──────────────────────────── 가장자리
alpha=0 (투명)      alpha=0→1 (점점 불투명)
```

**핵심 공식:**
1. 픽셀 좌표를 -1 ~ +1 범위로 정규화
2. 피타고라스 공식으로 중심까지의 거리 계산
3. 거리 0.5 이하 → 완전 투명 / 0.5~1.0 → 점점 불투명
4. 알파를 **제곱**해서 중앙은 거의 안 보이고 가장자리만 급격히 강조

> 텍스처는 **모양(마스크)** 역할만 한다. 실제 밝기는 `flashImage.color`가 제어한다.

---

## 두 색상이 합쳐지는 방식

Image 컴포넌트는 최종 화면에 픽셀을 그릴 때 텍스처 색과 `Image.color`를 **곱한다**.

```
최종 알파 = 텍스처 픽셀의 alpha  ×  flashImage.color.a
           (위치별 모양)           (시간에 따른 강도)
```

- 중앙 픽셀: `0 × 어떤값 = 0` → 항상 투명
- 가장자리: `1 × flash강도` → 피격 직후 최대, 시간이 지나면 사라짐

---

## 플래시 강도 감쇠 (Update)

타이머가 줄어들면서 알파값을 이렇게 계산한다:

```
normalizedTime = 남은시간 / 전체시간   (1.0 → 0.0)
decay          = normalizedTime²       (제곱으로 처음엔 천천히, 나중엔 빠르게 사라짐)
alpha          = 최대강도 × decay
```

`Time.unscaledDeltaTime`을 사용하기 때문에 처형 시간정지(timeScale=0) 중에도 정상 작동한다.

---

## 덮어쓰기 규칙 (Flash 메서드)

연속 피격 시 "더 강한 쪽"만 유효하다.

- 현재 남은 효과보다 **약한** Flash 호출 → 무시
- 현재 남은 효과보다 **강한** Flash 호출 → 덮어쓰기

---

## Inspector 파라미터 요약

| 파라미터 | 기본값 | 설명 |
|---|---|---|
| `flashImage` | — | 비네트 이미지가 붙은 UI Image 컴포넌트 |
| `maximumIntensity` | 1.0 | 알파 최대값 상한선 (1.0 = 완전 불투명까지 허용) |

> 강도(intensity)와 지속시간(duration)은 `DamageFlashListener`의 Inspector에서 조절한다.
